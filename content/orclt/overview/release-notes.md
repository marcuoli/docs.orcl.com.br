# Release Notes

Release notes and changelogs are published on the [GitHub Releases page](https://github.com/marcuoli/orclT/releases).

Each release includes:

- Pre-built binaries for Linux x86\_64 and Windows x86\_64
- SHA256 checksums for all artifacts
- A summary of new features, improvements, and bug fixes

## Where to Find Release Notes

Visit [github.com/marcuoli/orclT/releases](https://github.com/marcuoli/orclT/releases) to browse all published releases and download specific versions.
