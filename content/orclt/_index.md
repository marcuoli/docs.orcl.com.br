orclT is a unified command-line toolkit for Oracle Database administrators and operators. It consolidates the most common DBA tasks — from live performance monitoring and LOB analysis to patch downloads and backup management — into a single binary with a consistent interface.

Explore the sections below to get started with installation, learn about individual commands, and find the complete CLI flag reference.
